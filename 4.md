# 4. 두 실수를 메모리에 저장하고 함수 Add()에서 더한 후 리턴한 값을 화면체 출력하는 C언어 프로그램을 작성하시오.

```
#include <stdio.h>
double Add(double a, double b) {
    return a + b;
}
int main() {
    double x, y, z;
    scanf_s("%lf", &x);
    scanf_s("%lf", &y);
    z = Add(x, y);
    printf("%lf\n", z);
    return 0;
}
```
## 입력값 : 1.1, 2.2
## 출력값 : 3.300000
